Attribute Macro actix_web::test

#[test]
Available on crate feature macros only.
Expand description

Marks async test functions to use the Actix Web system entry-point.

§Examples

#[actix_web::test]
async fn test() {
    assert_eq!(async { "Hello world" }.await, "Hello world");
}
